PD-DEFINE-PATCH

[GENERIC-FUNCTION]

Function arguments:

  - NAME-OF-PATCH Define the fxp-presets to use in your sound. [default = nil]

Description:

It defines the fxp-presets to use in your sound with the object ckn-VST2. You need be careful because the binaries that I am using not accept very long paths. Then prefer smaller paths.
